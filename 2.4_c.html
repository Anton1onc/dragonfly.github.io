Dragonfly kernel v.2.4 Stable | Redmi 7

by @DfP_DEV | 21 March 2020


  • Upstream to 4.9.215
  • Removed support for AOSP Pie, renamed builds: MIUI and Q
  • Now a boot without a tweak module is impossible. So if you reflash,
    you must first start the system for the first time, and then install the kernel
  • The color of the display has become nicer
  • mproved smoothness and responsiveness
  • The smartphone now warms less and saves better
  • Removed smart mode (maybe in the future there will be something to replace)
  • The maximum display frame rate has been increased to 61
  • Support for dynamic FPS
  • Improved DfK application
  • Added two new profiles

2020, Dragonfly Project team
