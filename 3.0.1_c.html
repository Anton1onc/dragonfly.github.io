Dragonfly kernel v.3.0.1 Nightly | Redmi 7

by @DfP_DEV | 26 July 2020


  • Its own driver DStune, which responds not to touching the screen,
    but to the load. The values ​​that it sets correspond to the load on all cores combined,
    and there is support for hotplug (changes in the number of working cores "on the fly")
  • Improved sound DTS Eagle, as in the core Neon (thanks to its author)
  • Own I / O perf scheduler, which acts like a noop (100% performance),
    but does not allow any idle requests.
    Enabled in third-party tweakers in disk settings / schedulers
  • Overclocking CPU up to 2 GHz, GPU up to 825 MHz
  • Ultra undervoltage
